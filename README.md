# python-pytest-abs-sample-project

Sample test project

## Prepare environment

Using venv  

```sh
python -m venv env
env\Scripts\activate
(env) > pip install -r requirements.txt
```

## Run tests by markers using .bat file

Command: `run-tests.bat "{markers}" "{root-tests-folder}"`
